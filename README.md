# 文尼模板網 第43套

## 模板介紹
這是一款基於 **MacCMS V10** 的影視模板，適用於構建專業的影視網站。

- **橫圖版 Demo：** [wn43.wntheme.com](http://wn43.wntheme.com)
- **直圖版 Demo：** [wn43p.wntheme.com](http://wn43p.wntheme.com)

## 特點
- 支持橫圖與直圖兩種展示風格
- 響應式設計，適配各種設備
- 簡潔美觀，適合影視網站使用
- 高效的代碼結構，優化 SEO 表現
- 含主題後台
- 包含完整會員功能
- 支持文章功能

## 安裝指南
1. 將模板文件上傳到 MacCMS V10 的 `template` 目錄下。
2. 登錄後台，進入 **系統設置 -> 模板管理**。
3. 選擇上傳的模板並設置為默認模板。
4. 訪問1次域名後，模板會自動安裝
5. 刷新頁面查看效果
